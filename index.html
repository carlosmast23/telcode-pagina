Trabajo Seguro (Gratuita)

TrabajoSeguro permitirá publicar las necesidades de las personas, y los servicios de un trabajador mostrado su experiencia y conocimiento, una vez concluido el trabajo, las dos partes tendrán una calificación de esa manera se podrá recomendar a estas personas para futuros contratos. VER MAS

Simulador Exámenes Psicométricos

La aplicación esta orientada para escuelas de conducción que tiene que tomar las pruebas psicométricos, el software permite evaluar diferentes aspectos de cada participante, las pruebas son:Prueba de reacción.- Medir el tiempo de respuesta en presionar el pedal cuando el semáforo cambia de verde a rojo. Prueba de punteado.- La prueba cuenta con tres orificios y un disco giratorio el cual sólo permitirá visualizar un orificio a la vez, y con el mouse se tiene dar un clic en dichos orificios. Prueba de doble vía.- Se tiene que manejar dos carros a la vez utilizando un control de juegos y evitar salir fuera del camino hasta llegar a la meta.
